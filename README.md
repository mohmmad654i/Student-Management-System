# Student-Management-System
A simple C++ program to manage student information using structures
Student Data Management System
هذا المشروع عبارة عن نظام بسيط لإدارة بيانات الطلاب باستخدام لغة C++. تم بناؤه كجزء من رحلة التعلم في تخصص الـ IoT، مع التركيز على استخدام الـ Structs وتنظيم البيانات بشكل هيكلي.

🛠 الميزات (Features)
[x] إدخال بيانات الطالب (الإيميل، رقم الهاتف، الاسم، المعدل، والعمر).

[x] التحقق من صحة البيانات (التحقق من طول رقم الهاتف ونطاق العمر).

[x] هيكلية منظمة للبيانات باستخدام Nested Structs.

[x] إمكانية تحديث بيانات الطالب بعد الإدخال الأولي.

🚀 كيف تشغل الكود؟
تأكد من تثبيت مترجم C++ (مثل MinGW أو GCC).

قم بتحميل الملف main.cpp.

افتح الـ Terminal في المجلد واكتب:

Bash
g++ main.cpp -o student_system
./student_system
